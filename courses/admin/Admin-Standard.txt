[id:2/topic: The Four Important Tabs]

slide_id: 9ba9d651-8039-4737-b035-010dc54b1144
index_title: The Insights Tab
title: The Insights Tab
type: main
---
Open the Insights tab to browse all Insights included in your solution.

Each insight also shows a Data Source. This tells you where the data comes from. Inventory and Products pull from the Products source. Suppliers pull from Suppliers. Customers from Customers.

At the bottom of the list you will find Private Insights. These are only visible to you.
![video/size:large/align:center/frame:none](https://pub-3c39fac4c588469d9bf40bf9e56bdbaf.r2.dev/Admin/Admin-Insights-Tab.mp4)

index_title: The Data Tab
title: The Data Tab
type: main
---
If you are ever unsure what a column means or want to know exactly what data you are working with, open the Data tab.

Select a data source and you will see every available column, a description of what it contains, and a real example from your data. It is your reference point whenever something is unclear.
![image/size:large/align:center/frame:none](admin-standard/Data-Tab.png)
