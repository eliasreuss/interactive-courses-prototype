title: Public vs. Private
type: main
layout: standard-2-col
---
Before we begin, it’s important to understand the difference between public and private Insights, Actions, and Dashboards.

Everything in Inact Now is either public or private. Public content is visible to everyone in your organization, while private content is only visible to you.

Whenever you create something new, it starts as private. You choose when to share it.

Keep in mind that editing public content affects everyone who uses it. If you’re unsure, it’s always safest to work in private first.
![image/size:500px/align:right/frame:none](admin-custom/Public-Private.png)

[id:2/topic: Creating And Editing Insights]

slide_id: 6a283ef0-8b09-4da8-8f66-1bb6c6b80d45
index_title: Creating an Insight
title: Creating an Insight
type: main
---
The standard Insights are based on best practices and will cover most needs. But if you want something more tailored, you can create your own.

To create a new Insight, navigate to the Insights tab and click the “+ New Insight” button. This opens the template picker, where you can browse existing Insights and use one as the starting point for your own.

Select a template, give your Insight a name, add slicers and filters, and place it in an Insight Group. Then click Create. Your new Insight will be added to your library as a private Insight, ready for you to refine and share when you’re ready.
![video/size:large/align:center/frame:none](https://pub-3c39fac4c588469d9bf40bf9e56bdbaf.r2.dev/Admin/Create-Insight.mp4)

index_title: Edit Mode
title: Edit Mode
type: main
layout: standard-2-col
---
If you want to edit an existing Insight, or the Insight you just created, you can use "Edit Mode". Here you can add or remove columns, change KPIs, and adjust filters. When you save, you choose to keep it private or make it public.

As an Admin you can enter edit mode directly on any public insight. This is powerful, but be careful. Any change you make updates the insight for everyone in your organization immediately.

If you are experimenting, copy the insight first and work on the private copy.

If you want a step-by-step tutorial on how to use Edit Mode, click the dark green button below.
