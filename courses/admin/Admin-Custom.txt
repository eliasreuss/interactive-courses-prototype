index_title: Public and Private
title: Public and Private
type: main
layout: standard-2-col
---
Anything in Inact Now is either public or private. Public content is visible to everyone in your organization. Private content is only visible to you.

When you create something new, it starts as private. You decide when to make it public.

Editing public content affects everyone who uses it. If you are unsure, work in private first.

[id:2/topic: Creating And Editing Insights]

slide_id: 6a283ef0-8b09-4da8-8f66-1bb6c6b80d45
index_title: Creating an Insight
title: Creating an Insight
type: main
---
To create a new insight, click the "+ New Insight" button in the Insights tab. This opens a template picker where you can browse existing Insights to use as the foundation of your new Insight.

Select one, give it a name, add slicers and filters, organise it in an Insight Group and then click create. This will add it to your library as a private insight.
![video/size:large/align:center/frame:none](https://pub-3c39fac4c588469d9bf40bf9e56bdbaf.r2.dev/Admin/Create-Insight.mp4)
---

slide_id: 3e62aa68-bffb-4c4a-b390-5bd85da43fa7
index_title: Edit Mode
title: Edit Mode
type: main
layout: standard-2-col
---
If you want to edit an existing Insight, or the Insight you just created, you can use "Edit Mode". Here you can add or remove columns, change KPIs, and adjust filters. When you save, you choose to keep it private or make it public.

As an Admin you can enter edit mode directly on any public insight. This is powerful, but be careful. Any change you make updates the insight for everyone in your organization immediately.

If you are experimenting, copy the insight first and work on the private copy.

If you want a step-by-step tutorial on how to edit an Insight, click the dark green button below.
![image/size:550px/align:right/frame:none](admin-custom/Edit-Mode.png)
