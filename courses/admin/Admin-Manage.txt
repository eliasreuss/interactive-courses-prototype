The easiest way to find out if you have the admin panel, is to look for the icon in the top right corner of your screen. If it is there, you have access. If not, this course does not apply to you.

Your admin panel was set up by an Inact consultant. You might be a regular user, a super user, or an admin and still have it. It does not depend on your role.

[id:3/topic: Setting Up Subscriptions]

slide_id: 87aa6350-76ba-48e8-87bb-d8afce8b5344
index_title: Subscriptions - Mail Bundles
title: Subscriptions - Mail Bundles
type: main
---
The Subscriptions tab is where you set up automated email reports called mail bundles.

A mail bundle is a scheduled email that delivers a combination of Dashboards, Insights, or specific Insight Views directly to a recipient's inbox. You choose what is in it , when it sends, and who receives it.

Go through the tutorial below to learn how to set up subscriptions

slide_id: 4bbc4a91-e619-4c16-b025-357a4e900466
index_title: Subscriptions - Action Subscriptions
title: Subscriptions - Action Subscriptions
type: main
---
You can also set up Action subscriptions to alert users about updates on Action lists. First, select the Action you want to assign. Then choose when recipients should be alerted.

By default, recipients get an alert every time the Action has a new item. Tick "Action Subscription" to control the frequency instead. Tick "Alert Mail" to also send recipients update emails.

Finally, select your recipients and click "Start & Activate."
